# Getting Started

## Pre-requisites
npm - 9.6.2
node - v18.11.0


## Installation and setup
Make sure you have yarn installed:
```bash
npm install --global yarn
```
First, run the development server:
```bash
yarn install
```
```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.
